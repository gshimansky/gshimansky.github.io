How to clone a libvirt VM controlled by Vagrant

Strangely enough, googling about this problem doesn’t produce any meaningful results. So here are steps which allow cloning a libvirt VM controlled by Vagrant. Imagine that you have a VM named testvm1 which you want to clone into testvm2. Shutdown all VMs with vagrant halt or at least from inside of VM with sudo shutdown -h 0.
